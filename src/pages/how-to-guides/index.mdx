# How-To Guides

Welcome to the DevGuard how-to guides. These practical guides provide step-by-step instructions for common tasks and workflows to help you get the most out of DevGuard.

## Scanning

Integrate security scanning into your development workflow:

- [Scan OCI Images](/how-to-guides/scanning/scan-docker-images) — Scan container images for security issues.
- [Scan Source Code](/how-to-guides/scanning/scan-source-code) — Perform static analysis on your source code.
- [Scan Dependencies](/how-to-guides/scanning/scan-dependencies) — Scan your project dependencies for vulnerabilities.
- [Scan with GitHub Actions](/how-to-guides/scanning/scan-with-github-actions) — Integrate DevGuard scanning into GitHub Actions workflows.
- [Scan with GitLab CI](/how-to-guides/scanning/scan-with-gitlab-ci) — Integrate DevGuard scanning into GitLab CI/CD pipelines.
- [Upload SBOM](/how-to-guides/scanning/upload-sbom) — Upload Software Bill of Materials (SBOM) files to DevGuard.
- [Upload VEX](/how-to-guides/scanning/upload-vex) — Upload Vulnerability Exploitability eXchange (VEX) documents.
- [Scheduled Scans](/how-to-guides/scanning/scheduled-scans) — Configure automated, scheduled security scans.
