# `[METHOD] [/api/v1/endpoint/path]`

Brief description of what this endpoint does.

## Request

### HTTP Method
```
[GET | POST | PUT | PATCH | DELETE] /api/v1/endpoint/path
```

### Authentication

Required authentication method:
- üîê **PAT (Personal Access Token)** - Header: `Authorization: Bearer <token>`
- üîê **Session Cookie** - Cookie-based authentication

### Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `param1` | string | Yes | Description of parameter |
| `param2` | uuid | Yes | Description of parameter |

### Query Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `page` | number | No | 1 | Page number for pagination |
| `limit` | number | No | 20 | Items per page |
| `filter` | string | No | - | Filter criteria |

### Request Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Content-Type` | Yes | Must be `application/json` |
| `X-Asset-Ref` | No | Asset version reference |

### Request Body

```json
{
  "field1": "string",
  "field2": 123,
  "nested": {
    "field3": true
  }
}
```

**Schema:**

| Field | Type | Required | Description | Validation |
|-------|------|----------|-------------|------------|
| `field1` | string | Yes | Description | Max 255 chars |
| `field2` | number | Yes | Description | Min: 0 |
| `nested.field3` | boolean | No | Description | - |

## Response

### Success Response

**Status Code:** `200 OK` (or `201 Created`, `204 No Content`)

```json
{
  "id": "uuid",
  "field1": "value",
  "field2": 123,
  "createdAt": "2025-12-21T10:00:00Z",
  "updatedAt": "2025-12-21T10:00:00Z"
}
```

**Response Schema:**

| Field | Type | Description |
|-------|------|-------------|
| `id` | uuid | Unique identifier |
| `field1` | string | Description |
| `field2` | number | Description |
| `createdAt` | timestamp | ISO 8601 timestamp |
| `updatedAt` | timestamp | ISO 8601 timestamp |

### Pagination Response

For paginated endpoints:

```json
{
  "data": [...],
  "pagination": {
    "total": 100,
    "page": 1,
    "limit": 20,
    "totalPages": 5
  }
}
```

### Error Responses

```json
{

  "message": "field1 is required",
  "code": "VALIDATION_ERROR"
}
```

## Rate Limiting

- **Rate Limit:** 1000 requests per hour
- **Headers:**
  - `X-RateLimit-Limit`: Maximum requests per hour
  - `X-RateLimit-Remaining`: Remaining requests
  - `X-RateLimit-Reset`: Reset time (Unix timestamp)

<Callout type="warning">
  **Rate Limit Exceeded:** Returns `429 Too Many Requests`
</Callout>
