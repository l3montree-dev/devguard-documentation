For persistent issues that don't match common causes:

### Enable Debug Logging

```bash
# Enable verbose/debug mode
export DEBUG=true
command-with-debug-output
```

### Check System Resources

```bash
# Memory usage
free -h

# Disk space
df -h

# CPU usage
top
```

### Network Diagnostics

```bash
# Test connectivity
curl -v https://api.devguard.org/health

# Check DNS
nslookup api.devguard.org

# Test ports
telnet api.devguard.org 443
```

### Database Issues

```bash
# Check database connection

# Check database logs
```
